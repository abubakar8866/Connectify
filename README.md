Connectify – Full Stack Social Media Application

Connectify is a modern full stack social networking web application that enables users to connect, share posts, follow others, send direct messages, and receive real-time notifications.

The application is built using React.js (Frontend) and Spring Boot (Backend) with secure JWT authentication and real-time communication using WebSockets.                                                                        

Features
🔐 Secure Authentication (JWT)

👥 Role-Based Access Control (USER / ADMIN)

🖼 Image & Video Post Upload

❤️ Like & 💬 Comment System

👤 Follow / Unfollow Users

🔔 Real-Time Notifications

💬 Direct Messaging (One-to-One Chat)

📌 Saved Posts

🔎 Search Users & Posts

#️⃣ Hashtag Support

📊 Admin Dashboard

⚡ Optimistic UI Updates
